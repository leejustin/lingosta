# Lingosta API

## Prerequisites
1. [Install Vercel CLI](https://vercel.com/docs/concepts/functions/serverless-functions/quickstart):
   
   ```bash
   npm i -g vercel@latest
   ```

## Run Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server in the `@lingosta/` root directory:
   ```bash
   vercel dev
   ```
